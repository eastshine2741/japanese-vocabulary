# 학습 통계 (Study Stats) - User Stories

리텐션 향상을 목적으로 학습 통계 기능을 추가한다.

**현황 (이미 구현됨):**
- Words 탭: 복습할 단어 수(due cards), 단어 상태 분포(외운/외우는 중/새 단어), 학습하기 CTA

**설계 원칙:**
- Words 탭과 중복하지 않는다 — 카드/단어 현황은 이미 있다
- 새 기능은 **습관 형성**에 집중한다 (streak, 주간 패턴, 목표)
- 학습 맥락에 가까운 곳에 배치한다 — 통계는 학습 직후 보고 싶다

**배치 전략:**

| 화면 | 추가 요소 | 이유 |
|---|---|---|
| Home 탭 | Streak + 주간 미니 dots (compact card) | 매일 앱 열 때 첫인상. 최소 공간으로 최대 동기 부여 |
| Profile 탭 | 성취 지표 4개 + 주간 차트 + 일별 히트맵 | 학습 데이터 종합 뷰. 자기 프로필과 자연스럽게 묶임 |
| Settings | 일일 목표 설정 | 설정은 설정 화면에. 자주 바꾸는 값이 아님 |

**구현 우선순위:**

| 순위 | US | 기능 | 리텐션 기여 | 이유 |
|:---:|---|---|:---:|---|
| P0 | US-1 | 홈 화면 Streak + 주간 dots | 매우 높음 | 앱 열자마자 보이는 daily hook. Streak은 검증된 #1 리텐션 드라이버 |
| P1 | US-2 | 프로필 성취 지표 + 주간 차트 | 높음 | 학습 직후 피드백 루프. P0의 streak 데이터를 상세하게 보여주는 확장 |
| P1 | US-3 | 일별 학습 히트맵 | 높음 | 장기 학습 패턴 시각화. GitHub-style heatmap은 누적 성취감과 빈 칸 회피 동기를 동시에 만든다 |
| P2 | US-4 | 일일 목표 설정 | 보통 | 목표선이 있어야 주간 차트(P1)와 dots(P0)에 의미가 생김. 단, 없어도 streak만으로 동작 |

- **P0만 구현해도 리텐션 효과가 있다** — streak + dots는 독립적으로 동작
- P1은 P0의 데이터를 재활용하므로 추가 비용이 낮다
- P2는 P0/P1 없이는 의미가 없으므로 반드시 후순위

## US-1: 홈 화면 학습 스트릭 확인 `P0`

**As a** 학습자
**I want to** 앱을 열었을 때 연속 학습일과 이번 주 활동을 한눈에 보고 싶다
**So that** 매일 앱을 열고 학습을 이어갈 동기가 생긴다

**배치:** Home 탭, 검색 바와 최근 들은 노래 사이. compact card (~70px 높이)

### Acceptance Criteria

- [ ] 현재 연속 학습일(streak) 숫자가 아이콘과 함께 표시된다
- [ ] 이번 주(월~일) 7일의 학습 여부가 미니 dots로 표시된다
  - [ ] 학습한 날: 채워진 dot
  - [ ] 학습하지 않은 날: 빈 dot
  - [ ] 오늘: 테두리 등으로 구분
  - [ ] freeze 사용일: freeze 아이콘 dot
- [ ] 하루에 flashcard를 1장 이상 리뷰하면 해당 날짜가 학습일로 인정된다
- [ ] 하루 학습하지 않으면 streak이 0으로 리셋된다 (freeze 보유 시 자동 소모)
- [ ] Streak freeze
  - [ ] 7일 streak 달성 시 freeze 1회 적립된다
  - [ ] 최대 2회까지 보유할 수 있다
  - [ ] 학습하지 않은 날에 자동으로 1회 소모되어 streak이 유지된다
  - [ ] freeze 잔여 횟수가 카드에 표시된다
- [ ] 카드를 탭하면 Words 탭의 학습 통계 섹션으로 이동한다
- [ ] 기준 시간은 자정(KST)이다

## US-2: 프로필 학습 현황 상세 확인 `P1`

**As a** 학습자
**I want to** 프로필에서 주간 학습 패턴과 누적 성취를 한눈에 보고 싶다
**So that** 학습 패턴을 파악하고 부족한 날을 보완할 수 있다

**배치:** Profile 탭, 프로필 카드 아래

### Acceptance Criteria

- [ ] 성취 지표 4개가 한 줄에 표시된다
  - [ ] 현재 연속 (current streak, 일)
  - [ ] 최장 기록 (best streak, 일)
  - [ ] 총 학습일 (total study days)
  - [ ] freeze 보호 (보유/최대, 예: 1/2)
- [ ] streak milestone(7일, 30일, 100일) 달성 시 시각적 피드백이 표시된다
- [ ] "이번 주 활동" 섹션에 월~일 7일간 일별 리뷰 카드 수가 바 차트로 표시된다
  - [ ] 일일 목표선이 차트에 표시된다 (목표 라벨 포함, 예: "목표 10장")
  - [ ] 목표를 달성한 날은 시각적으로 구분된다
  - [ ] 오늘 날짜가 구분되어 표시된다 (라벨 "오늘")
  - [ ] 리뷰하지 않은 날은 "-"로 표시된다
  - [ ] freeze가 사용된 날은 freeze 아이콘과 함께 "보호"로 표시된다
- [ ] 차트 아래에 일별 리뷰 수 라벨이 요일과 함께 표시된다

## US-3: 일별 학습 히트맵 `P1`

**As a** 학습자
**I want to** 지난 몇 개월간 일별 학습 활동을 한눈에 보고 싶다
**So that** 장기 학습 패턴을 파악하고 빈 칸을 채우려는 동기가 생긴다

**배치:** Profile 탭, 주간 차트 아래 ("학습 기록" 섹션)

### Acceptance Criteria

- [ ] 일별 학습량을 격자(GitHub contribution 스타일) 히트맵으로 표시한다
- [ ] 학습량에 따라 5단계의 색상으로 구분된다
  - [ ] L0 (학습 안함): 가장 옅은 회색 (`#ECEEF2`)
  - [ ] L1~L4: accent-primary 계열로 농도가 점차 진해짐 (`#C8E1F5` → `#92C4E8` → `#5BA0D5` → `#2E7CB8`)
- [ ] 헤더에 "학습 기록" 타이틀과 누적 총 학습일 수가 표시된다 (예: "총 43일")
- [ ] 히트맵 하단에 범례가 우측 정렬로 표시된다 ("적음" → 5단계 색상 견본 → "많음")
- [ ] 일별 학습량 = 해당 날짜에 리뷰한 flashcard 수
- [ ] 색상 구간 임계치는 일일 목표(US-4) 대비 비율로 정한다 (예: 1~25%, 26~50%, 51~100%, 100%+)

---

## US-4: 일일 학습 목표 설정 `P2`

**As a** 학습자
**I want to** 하루에 리뷰할 카드 수 목표를 설정하고 싶다
**So that** 매일 달성감을 느끼며 꾸준히 학습할 수 있다

**배치:** Settings 화면, 학습 설정 섹션 내 (기존 FSRS/리뷰 시간 설정과 같은 그룹)

### Acceptance Criteria

- [ ] 일일 목표 카드 수를 설정할 수 있다
- [ ] 기본값은 10장이다
- [ ] 설정한 목표는 US-1의 홈 화면 dots, US-2의 주간 차트 목표선, US-3의 히트맵 색상 임계치에 반영된다
